# Smart Light Control System using ESP8266 and PIR Sensor

## Overview
A Wi-Fi based Smart Light Control System developed using ESP8266 NodeMCU and PIR Motion Sensor.

## Features
- Manual ON/OFF Control
- Automatic Motion Detection
- Wi-Fi Web Interface
- Real-Time Status Monitoring

## Components Used
- ESP8266 NodeMCU
- PIR Sensor (HC-SR501)
- LED
- 1kΩ Resistor
- Breadboard

## Software Used
- Arduino IDE
- Wokwi Simulator
- GitHub Desktop
